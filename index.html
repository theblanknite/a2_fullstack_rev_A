What does "Full" in full stack mean?

Full means that you are a full programmer, meaning that you can program on the client (user computer) end, and the server computer. The client computer is often referred to as the front end, and the server computer is referred to as the back end, so full means you can program in both the front and back end.

What does "Stack" in full stack mean?

The stack in full stack refers to the stack of programming languages that full programmers need to be familiar with. This includes front end programming languages such as HTML, CSS, Javascript and more, as well as back end languages such as PHP and mySQL. To be a full stack programmer, you will need to know these stacks of programs in order to develop both the front and back end.

What does LAMP Stack mean?

LAMP stands for:
L: Linux
A: Apache, which is our web server
M: MySQL, the database we use
P: PHP, which is the programming language we use on the server side

What is DB?

DB, or database, is an organized way to save data on a server. Databases of course are the lifeblood of all modern business, because they can be used for things like tracking employees, orders, customers, payrole, and many other important business related things.

What is MySQL?

MySQL is a widely used database. It is not the most common database, but still very commonly used under oracle. MySQL is FREE! The inventor of mySQL named this after his first daughter, and her name was "MY". This inventor later created MariaDB, after his second daughter, "Maria". Therefore MySQL is "My" + structured language query.

What is SQL?

SQL is a international standard language. It stands for structured Query Language. MySQL is the name of the database, while SQL is the language you use inside of that database to answer questions. SQL is used for many mainstream databases such as mySQL, Oracle, and Microsoft SQL.

What is PHP?

PHP is the programming language that connects front end to back end. It reads HTML forms and send info to a server. It also reads the database and sends that back to another HTML form. PHP is server side, and totally secure and private. PHP is the main back end language that makes communication between users and servers possible, and therefore makes eCommerce and eBusiness possible.
